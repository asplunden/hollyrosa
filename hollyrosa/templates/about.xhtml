Hollyrosa

Hollyrosa is a program booking system built for the program crew at the Vässarö Scout Centre. The original purpose of Hollyrosa is to handle all program bookings for the eight week summer season but is evolving to handle the whole year and also handling staff, course and group data for general booking and scheduling. Hollyrosa is designed to enhance communication and visualization. Hollyrosa was first tested in the summer of 2010 and that first fligth was very successfull, we now have eight years of experience running the system. Most of what we learned using the system has been incorporated along the way.

Technology and Thanks

Hollyrosa uses the Turbogears2 (TG2) mega framework for building database backed websites In Python. It contains a bunch of frameworks you don't want to miss.

Hollyrosa uses the Dojo JavaScript framework for a lot of dynamic action.

The database is CouchDB (it used to be either SQLite or MySQL. SQLAlchemy handles both and a few more I haven't tested.)

Hollyrosa has been adapted to use BULMA css framework.
