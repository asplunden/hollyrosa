Hollyrosa Program Booking System

Welcome to Hollyrosa program booking system

This is the booking system for program at Vässarö.

Calendar Overview


View the calendar overview of the whole summer. From the calendar overview, you may select a day to see and work with all bookings for that day.

Visiting Groups


List (all) visiting groups. From the view of all visiting groups, find the visiting group you are looking for or add a new group. You may edit, show and list activities of the visiting group you are intereseted in.

New booking request


Add a new booking request, it is often more effective to first go to the list of visiting groups, then under the group the booking request is for, select new booking request from the mini-menu.
