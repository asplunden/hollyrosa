Hollyrosa - About

About Hollyrosa

Hollyrosa is a program booking system built for the program crew at the Vässarö Scout Centre. The purpose of Hollyrosa is to handle all program bookings for the eight week summer season. Hollyrosa is designed to enhance communication and visualization of all bookings. Hollyrosa was first tested in the summer of 2010 and first fligth was very successfull. What we learned has been incorporated for the summer of 2011 edition of Hollyrosa currently in use.

Right now, around Christmas of 2011, the 2012 edition of the system is developed and the big change is that the whole database system is switched ovet to CouchDB. It's a big change that will enable us to operate the booking system on aa simple laptop even if the internet is down and the power is gone, things that happen from time to time in the archipelago.

Hollyrosa is actively being developed by Martin Eliasson.

Technology and Thanks

Hollyrosa uses the Turbogears2 (TG2) mega framework for building database backed websites. In Python. It contains a bunch of frameworks you don't want to miss. Especially not SQLAlchemy.

Hollyrosa uses the Dojo JavaScript framework for a lot of dynamic action.

The database is CouchDB (it used to be either SQLite or MySQL. SQLAlchemy handles both and a few more I haven't tested.)

Thanks to Jossan for patiently testing the system for real.
